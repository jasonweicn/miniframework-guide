# 目录结构

GitBook allows you to organize your book into chapters, each chapter is stored in a separate file like this one.

```
MiniFramework/
|--- App/                   应用案例
|    |--- Api/              REST模式的API
|    |--- Cache/            缓存
|    |--- Config/           配置
|    |    |--- database.php 数据库配置文件
|    |
|    |--- Controller/       控制器
|    |--- Layout/           布局
|    |--- Model/            模型
|    |--- Public/           站点根目录
|    |    |--- css/         css
|    |    |--- img/         图片
|    |    |--- js/          js
|    |    |--- index.php    应用入口文件
|    |
|    |--- View/             视图
|
|--- MiniFramework/         框架核心目录
     |---Functions/         函数库
     |---Library/           类库
     |---Bootstrap.php      引导程序
```
